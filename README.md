### Start

```
npm i -g react-native-cli
```

```
npm i
```

```
npm run ios
# or
npm run android
```

### Build ios

1.bundle js files

```
npm run bundle-ios
```

2.Xcode: 

    2.1 Add `js bundle files` to `GithubApp` project
    
    2.2 Xcode –> Product –> Scheme –> Edit Scheme…
    
        info -> Build Configuration: Release

    2.3 Build
    
### Build android

```
cd android
```

```
./gradlew assembleRelease
```
